Introducción a HTML y CSS

HTML (HyperText Markup Language) es el estándar para crear y diseñar páginas web. CSS (Cascading Style Sheets) se utiliza para estilizar y diseñar visualmente los documentos HTML.

HTML: Estructura Básica

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>Encabezado Principal</h1>
    <p>Este es un párrafo.</p>
    <a href="https://www.ejemplo.com">Enlace</a>
  </body>
</html>

HTML: Elementos Comunes

Encabezados

<h1>Encabezado Nivel 1</h1>
<h2>Encabezado Nivel 2</h2>
<h3>Encabezado Nivel 3</h3>

Listas

<ul>
  <li>Elemento de lista</li>
  <li>Otro elemento</li>
</ul>

<ol>
  <li>Elemento numerado</li>
  <li>Otro elemento</li>
</ol>

Tablas

<table>
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Celda 1</td>
    <td>Celda 2</td>
  </tr>
</table>

CSS: Selección de Elementos

Selección por Etiqueta

p {
  color: blue;
}

Selección por Clase

.mi-clase {
  font-size: 20px;
}

Selección por ID

#mi-id {
  background-color: yellow;
}

CSS: Propiedades Comunes

Colores y Fuentes

body {
  color: #333;
  font-family: Arial, sans-serif;
}

Margenes y Rellenos

.box {
  margin: 10px;
  padding: 20px;
  border: 1px solid #ccc;
}

Estilización de Enlaces

a {
  color: red;
  text-decoration: none;
}

a:hover {
  text-decoration: underline;
}

CSS: Layout y Diseño

Box Model

.box {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid black;
  margin: 15px;
}

Flexbox

.container {
  display: flex;
  justify-content: space-between;
}

.item {
  width: 100px;
  height: 100px;
  background-color: lightblue;
}

Grid

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}

.grid-item {
  background-color: lightgreen;
  padding: 20px;
}

HTML y CSS Avanzado

Transiciones y Animaciones

.box {
  width: 100px;
  height: 100px;
  background-color: coral;
  transition: background-color 0.5s ease;
}

.box:hover {
  background-color: blue;
}

Estilos Responsivos

@media (max-width: 600px) {
  body {
    background-color: lightgray;
  }
}

Variables CSS

:root {
  --main-color: #3498db;
}

.box {
  background-color: var(--main-color);
}
